## 2.📖 Project Descriptions
1. ATM Simulator (atm.py)
Features a menu-driven interface to manage virtual bank accounts, including PIN verification and transaction history logic.

2. Digital Clock (clock.py)
Utilizes time-based modules to fetch and refresh the system time, displaying a live clock in the console.

3. Number Guessing Game
A game that picks a random number within a range and provides feedback (higher/lower) to the user until they guess correctly.

4. Rock, Paper, Scissors
Implements logic to compare user input against the computer's randomized choice using conditional statements.

5. Pascal's Triangle (pascal.py)
Uses nested loops and combinations logic to generate the rows of Pascal’s Triangle.
